# Frontend Auth Example

## Setup

1. Copy `.env.local.example` to `.env.local` and fill in your Auth0 credentials.
2. Install dependencies:
```bash
npm install
```
3. Run dev server:
```bash
npm run dev
```

## Auth Routes
- `/api/auth/login`
- `/api/auth/logout`
- `/api/auth/callback`
- `/api/auth/me`

## Landing Page
Accessible at `/` with a Login button.
